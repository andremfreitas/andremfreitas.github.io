# AGENTS.md

This repository contains a personal website built from an al-folio/Jekyll-style GitHub Pages project.

## Working Guidelines

- Inspect the existing Liquid, Jekyll, Sass, and asset conventions before making changes.
- Keep edits small and focused on the requested behavior.
- Avoid unnecessary dependency, formatting, or generated-file changes.
- Do not add `polyfill.io` back to the site. Avoid untrusted third-party scripts unless they are explicitly needed and reviewed.
- Completed repository changes should always be committed and pushed to the configured remote.
